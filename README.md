### the problem

sharing a gitlawb repo today looks like this:

```
did:gitlawb:z6MkqDnb7S2VcFx38Xg2SvFV2ENsu3dLDoMkv8j2H4kRpQwLm9xN/awesome-repo
```

unreadable. unshareable. ugly in tweets and READMEs.

### the fix

```
gitlawb.link/abc
```

one short URL. one preview card. one click to the repo.

---

### what's inside

| feature | description |
|---------|-------------|
| 🔗 **short URLs** | turn long gitlawb DIDs into clean, memorable links |
| 🎴 **preview cards** | auto-generated OG images for every repo |
| 📎 **embed code** | one-liner to put a live repo badge in any README |
| 📊 **click analytics** | track how your shared links perform |
| 🌐 **open graph** | beautiful unfurls on Twitter, Discord, Telegram |
